# Claude Storyteller

An AI-powered RimWorld storyteller that uses Claude to make narrative decisions based on your colony's actual state.

## Features

- Analyzes colony wealth, defenses, mood, food, and recent history
- Three event cycles: minor, major, and narrative arcs
- Difficulty-aware — higher settings unlock more chaotic AI behavior
- Exploits actual weaknesses, not just random events

## Installation

1. Subscribe on [Steam Workshop](#) (link coming soon)
2. Or clone this repo into your RimWorld/Mods folder
3. Requires [Harmony](https://steamcommunity.com/sharedfiles/filedetails/?id=2009463077)

## Setup

1. Get API key from [console.anthropic.com](https://console.anthropic.com)
2. In-game: Mod Settings → Claude Storyteller → Paste key → Test Connection
3. Start new game → Select "Claude AI" storyteller

## Cost

Uses Anthropic's Claude API. ~$0.01-0.05 per hour of gameplay.
